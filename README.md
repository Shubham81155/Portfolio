# Shubham Pandey - Data Analyst Portfolio

A modern, responsive personal portfolio website showcasing analytics projects, dashboards, and data skills.

## Live Repository

- GitHub: [Portfolio](https://github.com/Shubham81155/Portfolio)

## Project Overview

This portfolio highlights selected work in:

- Power BI dashboard development
- Python-based analytics and machine learning projects
- Data storytelling and business insight communication

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Features

- Responsive, mobile-friendly layout
- Animated hero and section transitions
- Project showcase cards with links
- Skills section with visual bars
- Contact form UI

## Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Shubham81155/Portfolio.git
```

2. Open the project folder:

```bash
cd Portfolio
```

3. Open `index.html` in your browser.
